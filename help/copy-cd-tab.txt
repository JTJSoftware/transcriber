The Copy CD tab has fields for entering CDDA2WAV parameters to copy  a music cd. 
There is field at the bottom of the page that shows what the assembled parameters look like that will be passed to the CDDA2WAV program. You update this field by pressing the Refresh copystring button. The Refresh copystring button assembles the information you have entered in the above fields. The control string may be edited directly if needed. All of the entry fields in this tab have balloon help popups. The Rip Tracks NOW button copys the cd according to the parameters that are shown in the control string field.
